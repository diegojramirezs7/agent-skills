## CLI tool that helps you start any project
- asks you to choose only backend, only frontend, both
- choose framework
- asks framework based decisions: what kind of router, what kind of ORM, admin panel, etc
- Set up pre-commits (pre-push) hooks
- Add references (Commit messages, best practices for the framework, processes, etc.)
- Add design systems (org based)
- component libraries
- claude code skills
- cursor/vs code setup
